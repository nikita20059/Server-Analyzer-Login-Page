
## Project Description

Server Analyzer is a web application used to monitor and analyze server performance.
It helps administrators track server status, manage data, and analyze system activity through a user-friendly dashboard.

## Technologies Used

* React JS (Frontend)
* Node.js (Backend Runtime)
* Express.js (Backend Framework)
* MongoDB (Database)
* REST API

## Features

* Secure Login Page
* Server Monitoring Dashboard
* Performance Analysis
* Data Management
* Responsive User Interface

## Installation

1. Clone the repository
2. Install dependencies

Frontend:
npm install

Backend:
npm install

3. Run the project

Frontend:
npm start

Backend:
node server.js
